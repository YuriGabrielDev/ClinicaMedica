# Sistema de Clínica Médica em Java com Interface e Banco de Dados

Projeto desenvolvido como **trabalho escolar**, com o objetivo de aplicar na prática conceitos de programação em Java, interface gráfica e integração com banco de dados.

O projeto também é utilizado como **portfólio pessoal** e **estudo livre**, demonstrando conhecimentos básicos de desenvolvimento de sistemas.

## Objetivos do Projeto
- Praticar programação orientada a objetos em Java
- Desenvolver uma interface gráfica funcional
- Realizar operações básicas em banco de dados
- Treinar a capacidade de resolver um problema por meio da abstração

## Funcionalidades
- Cadastro e manipulação de dados de médicos, secretárias e pacientes

## Tecnologias Utilizadas
- Java

## Como executar
1. Certifique-se de ter o Java instalado
2. Execute a aplicação pela IDE ou via terminal

## Observações
- Projeto com fins educacionais
- Código aberto para estudo
